# RiskLens AI
AI-Assisted Private Credit Risk Dashboard

RiskLens AI is a modern credit risk monitoring dashboard designed to simulate how underwriting teams analyze borrowers, stress test financials, and generate decision memos.

This project demonstrates how AI workflows, financial analytics, and modern web development can combine to support real-world private credit decisioning systems.

# The application allows users to:

Review borrower risk metrics

Simulate financial stress scenarios

Generate AI-assisted underwriting memos

Visualize credit health indicators

Demonstrate explainable risk scoring

This project was built to showcase how software engineering and finance domain knowledge can integrate into real operational tools.

# Core Features
Borrower Risk Dashboard

Displays borrower credit profiles including:

Revenue

EBITDA

Debt-to-Income

DSCR

Credit utilization

Risk score classification

Stress Testing Engine

Users can simulate:

Revenue shocks

Margin compression

The system recalculates:

Stressed DSCR

Adjusted risk score

Updated underwriting recommendation

Example outputs:

Approve
Conditional Approval
Decline / Restructure

AI-Generated Credit Memo

# The dashboard produces an explainable memo summarizing:

base risk assessment

stress scenario results

underwriting recommendation

analyst commentary

This simulates how AI could assist credit analysts in drafting underwriting reports.

# Example Workflow

Select a borrower from the monitoring queue

Review financial metrics and current risk score

Run a stress scenario (sales decline / margin compression)

Analyze stressed DSCR and updated risk score

Generate an AI-assisted underwriting memo

Export memo or send for credit committee review
